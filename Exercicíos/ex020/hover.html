Exemplos de Hover

O que são Hovers em CSS?

Hover é uma Pseudo-class em CSS. Pseudo-class
são estados especiais de elementos no documento HTML.
E são especificados em seletores CSS. como no exemplo abaixo (em código e prático). 


                    
<html>

<style>
div {
    background-color: gray;
        width: 200px;
        height: 200px;
        text-align: center;
        line-height: 100px;
        border: solid 1px black;
        margin: auto;
}

div:hover {
    background-color: red;
    border-radius: 200px;
    color: white;
    font-size: 2em;
}
</style>

<div>
     div teste
</div>
</html>
